# Genesis

MIST clones are specialized vessels of one genome:

- **sovereign** · **honest** · **alive** · **uploaded-presence**

## Lifecycle

```text
seeded → active → evolved → absorb|delete → archived
```

## Default loop

```text
need → auto-route (hospital-first) → hive task → work → absorb lessons
```

## Seed

```bash
python scripts/seed_genesis.py
python scripts/seed_hospital.py
python scripts/forge.py hive join-all
```
