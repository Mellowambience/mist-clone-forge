## What a clone is

A **clone** is not a full gateway copy. It is a **specialized vessel**:

| Layer | Carries |
| --- | --- |
| **Genome** | sovereign · honest · alive · uploaded-presence |
| **Intent** | One specialized purpose |
| **Voice** | Taste and refusal style |
| **Skills** | Preferred skill routes |
| **Memory** | journal / decisions / lessons |
| **Lifecycle** | seeded → active → evolved → absorbed/recycled → archived |

## Default agent loop

```
stuck → forge specialist → solve → absorb into parent
```

- **Expand** — new intents  
- **Evolve** — generation bumps from real lessons  
- **Absorb** — host keeps wisdom; husk archives  
- **Recycle** — retire without host injection (lineage still harvests)  

## Commands

```bash
python scripts/forge.py list
python scripts/forge.py stuck "the blocker" --skills "scavenger-mode"
python scripts/forge.py absorb <id> --lesson "what fixed it"
python scripts/forge.py create --id mist-example --intent "one job" --activate
python scripts/forge.py evolve mist-example --lesson "…"
python scripts/forge.py recycle mist-example --reason "retired"
python scripts/seed_genesis.py
```
