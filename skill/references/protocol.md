# Clone protocol (quick card)

## States

```
seeded → active → evolved → absorb|recycle → archived
```

## Stuck → forge → absorb

```
stuck? ──yes──► forge.py stuck "blocker"
                    │
                    ▼
              work as clone
                    │
                    ▼
         forge.py absorb <id> --lesson "…"
                    │
                    ▼
         host wiser · husk archived
```

## Absorb checklist

- [ ] Blocker actually fixed (or abandoned with lesson)  
- [ ] Final `--lesson` written  
- [ ] `--into` set if parent is wrong host  
- [ ] Host journal/lessons updated  
- [ ] Donor gone from `clones/`, present in `archive/`  

## Forge checklist

- [ ] Intent is one sentence and falsifiable  
- [ ] Domain set  
- [ ] Parent exists (or is mist-prime)  
- [ ] ≤6 skills  
- [ ] Ephemeral if stuck-helper  

## Recycle checklist (no host)

- [ ] Reason written  
- [ ] Lessons in lineage  
- [ ] Prefer absorb when a host should grow  

## Provenance

| Tag | Meaning |
| --- | --- |
| `local` | Ran/read on machine |
| `retrieved` | From clone journal / MIST files |
| `inferred` | Reasoned; mark low confidence |
| `unknown` | Do not present as fact |
